The programs (apps) you need to install depend on how you intend to use the system.

## 1. General Use (Guide & Tourist)
**NO Installation Required!**

*   **Target**: Guides connecting to a PC server, or Tourists.
*   **Requirements**: Default Web Browser (Chrome, Safari, Samsung Internet, etc.).
*   **How to use**: Simply access the URL provided by the server administrator.
*   **Note**: For Guides (speakers), **Google Chrome** is recommended for the best Speech-to-Text performance.

## 2. Hosting Server on Phone (Android Only)
You only need to install an app if you want to **turn your phone into the Broadcast Station (Server)** without a PC.

*   **Target**: Users who want to run the guide system solely on a smartphone.
*   **App to Install**: **Termux**
    *   Download from: Google Play Store or F-Droid.
*   **Purpose**: To set up a Python environment on the phone to run the server.
*   **Usage**: Refer to `MOBILE_GUIDE_EN.md` for detailed instructions.
