# 📱 핸드폰 설치 프로그램 가이드

핸드폰 사용 용도에 따라 설치해야 할 프로그램이 다릅니다.

## 1. 일반적인 경우 (가이드 및 관광객)
**아무것도 설치할 필요가 없습니다! (No App Required)**

*   **대상**: PC에서 서버를 켜고 접속하는 가이드, 또는 모든 관광객
*   **준비물**: 기본 웹 브라우저 (Chrome, Safari, 삼성 인터넷 등)
*   **방법**: 서버 관리자가 보내준 주소(URL)로 접속만 하면 됩니다.
*   **참고**: 가이드(말하는 사람)는 원활한 음성 인식을 위해 **Google Chrome** 브라우저 사용을 권장합니다.

## 2. 폰을 "서버"로 쓰고 싶은 경우 (안드로이드 전용)
PC 없이 **내 핸드폰을 방송국(서버)으로 만들고 싶은 경우**에만 앱 설치가 필요합니다.

*   **대상**: 노트북 없이 핸드폰만으로 가이드 시스템을 운영하려는 분
*   **설치할 앱**: **Termux**
    *   다운로드: 구글 플레이 스토어 또는 F-Droid
*   **용도**: 핸드폰에 Python 실행 환경을 구축하여 서버를 구동합니다.
*   **사용법**: 상세한 사용법은 `MOBILE_GUIDE.md` 파일을 참고하세요.

## 요약표

| 역할 | 설치할 앱 | 비고 |
|------|-----------|------|
| **관광객 (듣기)** | ❌ 없음 | 브라우저로 접속 |
| **가이드 (PC 서버 사용)** | ❌ 없음 | 브라우저로 접속 |
| **가이드 (폰 서버 구동)** | ⭕ **Termux** | 안드로이드만 가능 |
