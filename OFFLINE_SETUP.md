# ✈️ 해외/오프라인 환경 무선 가이드 시스템 구축 가이드

인터넷이 없는 해외 현지에서 **사설 무선 공유기(AP)**를 활용하여 가이드 시스템을 구축하는 방법입니다. 인터넷 연결 없이 **로컬 네트워크(LAN)**만으로 동작합니다.

## 1. 하드웨어 준비물 (Hardware)

1.  **무선 공유기 (Wireless Router)**
    *   **필수 사양**: 전원 공급이 원활할 것 (보조배터리 연결 가능 모델 추천)
    *   **추천**: 휴대용 여행 공유기 (예: TP-Link, GL.iNet 등) 또는 일반 공유기 + 보조배터리
    *   **설정**: 인터넷 선(WAN) 연결 없이 전원만 켜서 사용합니다.
2.  **노트북 (서버용 PC)**
    *   가이드 서버 프로그램(`server.py`)을 실행할 컴퓨터입니다.
    *   공유기와 유선(LAN 케이블) 또는 무선으로 연결합니다.
3.  **스마트폰 (가이드 및 관광객)**
    *   가이드용 1대 + 관광객용 N대

## 2. 사전 준비 (한국에서 미리 할 것)

인터넷이 안 되는 환경에서는 패키지 설치가 불가능하므로, **반드시 출국 전에 세팅을 완료**해야 합니다.

1.  **서버 프로그램 세팅 완료**:
    *   `pip install -r requirements.txt` 까지 모두 완료해두세요.
    *   한 번 이상 실행해서 잘 되는지 확인하세요.
2.  **공유기 와이파이 설정**:
    *   SSID (와이파이 이름)와 비밀번호를 미리 설정해두세요. (예: `Guide_Tour`, `12345678`)

## 3. 현지 설치 및 실행 순서

### 1단계: 네트워크 구축
1.  **공유기 전원 켜기**: 인터넷 선은 꽂지 않습니다. 전원만 켭니다.
2.  **서버 PC 연결**: 노트북을 공유기 와이파이에 연결합니다. (또는 랜선 연결)
3.  **IP 주소 확인**:
    *   노트북에서 `cmd` -> `ipconfig` 입력.
    *   `무선 LAN 어댑터` 또는 `이더넷 어댑터`의 **IPv4 주소**를 적어둡니다. (예: `192.168.0.5`)

### 2단계: 서버 실행
1.  노트북에서 터미널(CMD)을 열고 서버를 실행합니다.
    ```bash
    python server.py
    ```
    *   *주의: 인터넷이 없으므로 ngrok은 실행하지 않습니다.*

### 3단계: 접속 (Connect)
1.  **스마트폰 와이파이 연결**:
    *   가이드와 모든 관광객의 폰을 **준비한 공유기 와이파이**에 연결합니다.
    *   *참고: "인터넷 연결 없음" 메시지가 떠도 "연결 유지"를 선택해야 합니다.*
2.  **브라우저 접속**:
    *   스마트폰 브라우저 주소창에 아까 적어둔 **노트북 IP 주소**와 포트를 입력합니다.
    *   주소 예시: `http://192.168.0.5:5000`
3.  **가이드/관광객 모드 선택**:
    *   평소와 같이 각자 역할을 선택하고 방송을 시작합니다.

## 4. 주의사항 (Limitations)

1.  **음성 인식(STT) 및 번역 제한**:
    *   Google 음성 인식(Chrome STT)과 번역 기능은 **인터넷이 필요**합니다.
    *   **오프라인 환경**에서는 **오디오 방송(목소리 듣기)**만 가능합니다. 자막과 번역은 작동하지 않을 수 있습니다.
2.  **안드로이드 마이크 권한**:
    *   최신 안드로이드/iOS 보안 정책상 `http://` (보안되지 않은 연결)에서는 마이크 권한을 막을 수 있습니다.
    *   **해결책**:
        *   노트북 브라우저 주소창(chrome://flags)에서 `Insecure origins treated as secure` 설정에 `http://192.168.x.x:5000`을 추가해야 할 수도 있습니다 (가이드 폰).
        *   또는 가이드만 **노트북 마이크**를 사용하여 방송하는 것을 권장합니다.
